# METSFlask  

**NOTE: Very much in dev**  

Flask application for viewing detailed information on "original" files in an Archivematica METS file.  

## Install (dev)
* Clone repo
* Create py3 venv and activate
* pip install Flask   
* pip install lxml  
* `flask run`
